# Statistical Data Analysis Project

📌 Project Overview
This project focuses on exploratory statistical data analysis to identify patterns and relationships within data. The objective was to apply statistical techniques learned in the IIT Madras BS Data Science program.

📚 Key Concepts Applied
- Descriptive Statistics
- Data Distribution Analysis
- Statistical Visualization
- Variable Relationship Analysis

📊 Key Analysis Performed
- Measures of central tendency
- Measures of variability
- Data visualization and charts
- Interpretation of statistical patterns

🛠 Tools Used
- Google Sheets / Excel
- Statistical Techniques
